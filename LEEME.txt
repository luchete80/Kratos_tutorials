TO BUILD KRATOS:

1 - Remember to set first KRATOS_SOURCE env bar: export KRATOS_SOURCE=/path/to/kratos/
2 - Run /path/to/kratos/scripts/standard.configure, changing some things (see below)
3 - Run make install in linux()
